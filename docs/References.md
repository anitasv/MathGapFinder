# References

Recommended study material for each course in [Courses.MD](Courses.MD). For each course, we suggest:

- **Textbook(s)** — standard references.
- **Course(s)** — freely available lecture notes / video lectures from MIT OpenCourseWare (OCW), Princeton, Harvard, Stanford, etc.
