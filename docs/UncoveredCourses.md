# Uncovered Courses

The current `docs/Courses.MD` lists 23 courses covering algebra, analysis,
topology, and number theory, with only a single "Probability" entry. A large
portion of the standard high-school-through-graduate mathematics curriculum is
missing. Below are the standard (non-specialized) courses that should be added
to broaden coverage.

## High School

- **Geometry** — Euclidean plane and solid geometry (Pythagoras, similar
  triangles, circle theorems, etc.)
- **Trigonometry** — identities, laws of sines and cosines
- **Statistics & Probability (introductory)** — descriptive statistics,
  counting, elementary probability

## Undergraduate

- **Calculus III / Vector Calculus** — Green's, Stokes', and Divergence
  theorems (the existing Multivariable Calculus tag does not surface these)
- **Ordinary Differential Equations (ODE)**
- **Partial Differential Equations (PDE)** — introductory: heat, wave, Laplace
- **Probability Theory** — a proper course beyond the single existing
  "Probability" tag (LLN, CLT, conditional expectation, etc.)
- **Mathematical Statistics** — estimators, Cramér–Rao, Neyman–Pearson,
  hypothesis testing
- **Discrete Mathematics** — logic, sets, induction, basic combinatorics
- **Combinatorics** — generating functions, Ramsey theory, inclusion–exclusion
- **Graph Theory** — Euler/Hamilton, Menger, Kuratowski, chromatic theorems
- **Numerical Analysis** — interpolation, quadrature, fixed-point and Newton
  convergence
- **Differential Geometry of Curves and Surfaces** — Frenet–Serret,
  Gauss–Bonnet
- **Set Theory & Mathematical Logic** — Cantor's theorem, Schröder–Bernstein,
  compactness, completeness
- **Mathematical Optimization / Linear Programming** — LP duality, KKT,
  simplex method

## Graduate

- **Measure-Theoretic Probability** — martingales, Kolmogorov's theorems
- **Stochastic Processes** — Markov chains, Brownian motion, basic Itô
  calculus
- **Mathematical Statistics (graduate)** — asymptotics, decision theory
- **Partial Differential Equations (graduate)** — Sobolev spaces, weak
  solutions
- **Ordinary Differential Equations / Dynamical Systems (graduate)** —
  existence–uniqueness, stability, Poincaré–Bendixson
- **Differential Geometry / Riemannian Geometry** — manifolds, connections,
  curvature
- **Galois Theory** — often part of Algebra, but worth a dedicated tag
- **Representation Theory** — of finite groups; Maschke, Schur
- **Algebraic Number Theory** — rings of integers, Dirichlet unit theorem,
  class number
- **Analytic Number Theory** — Prime Number Theorem, Dirichlet's theorem on
  primes in arithmetic progressions
- **Harmonic / Fourier Analysis** — Plancherel, Hausdorff–Young,
  Riesz–Thorin
- **Category Theory** — Yoneda lemma, adjoint functor theorem
- **Mathematical Logic / Model Theory / Computability** — Gödel
  incompleteness, Löwenheim–Skolem, halting problem
- **Convex Analysis / Optimization (graduate)** — Hahn–Banach separation,
  Fenchel duality
- **Numerical Linear Algebra** — QR, SVD, conditioning

All entries above are standard textbook subjects, not narrow specializations
(e.g., étale cohomology, operator algebras, geometric group theory are
intentionally omitted).
